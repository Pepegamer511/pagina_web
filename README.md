# AuraAcademy - Plataforma Educativa Web

## Descripción del Proyecto

AuraAcademy es una plataforma web educativa desarrollada con HTML, CSS y JavaScript, enfocada en la gestión académica y administrativa de cursos tecnológicos especializados. El sistema fue diseñado con una interfaz moderna e interactiva que permite simular el funcionamiento de una academia virtual orientada a áreas como modelos predictivos y algoritmos genéticos.

## Funcionalidades Implementadas

### Página Pública
Se desarrolló una landing page informativa donde los usuarios pueden:

- Visualizar la oferta académica disponible.
- Conocer los beneficios de la plataforma.
- Explorar programas educativos especializados.
- Solicitar inscripción mediante formularios interactivos.
- Acceder al sistema mediante un módulo de inicio de sesión.

### Sistema de Administración
Se implementó un panel administrativo dinámico que incluye:

- Gestión de estudiantes y solicitudes de inscripción.
- Aprobación o rechazo de aspirantes.
- Control de alumnos matriculados.
- Estadísticas básicas del sistema.
- Administración de cursos activos.
- Seguimiento del progreso académico.
- Configuración del perfil del administrador.
- Gestión de parámetros del sistema y notificaciones.

### Funcionalidades Dinámicas con JavaScript
El proyecto incorpora múltiples interacciones dinámicas como:

- Apertura y cierre de modales.
- Validación de acceso de administrador.
- Registro dinámico de nuevos estudiantes.
- Actualización automática de contadores y estados.
- Navegación mediante pestañas dentro del panel administrativo.
- Simulación de gestión académica en tiempo real.

## Tecnologías Utilizadas

- HTML5
- CSS3
- JavaScript
- Font Awesome

## Objetivo del Proyecto

El objetivo principal de este proyecto es simular una plataforma educativa moderna con herramientas administrativas interactivas, aplicando conceptos de desarrollo web frontend, diseño responsive y manipulación dinámica del DOM utilizando JavaScript puro.
